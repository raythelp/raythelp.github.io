# 影片說明文件

## videos/ - 影片資料夾

### intro.mp4 - 首頁開場影片
這個影片會在首頁自動播放，作為網站的開場 Vlog。

**建議規格：**
- 解析度：1920x1080px (Full HD)
- 長度：15-30 秒
- 格式：MP4 (H.264 編碼)
- 檔案大小：建議 10MB 以下
- 內容：東石鄉相關的精彩片段

**建議內容：**
- 東石漁人碼頭的日落美景
- 鰲鼓濕地的候鳥生態
- 蚵田養殖的漁村風光
- 新鮮蚵仔料理的製作過程
- 漁船進出港的活力景象
- 傳統市集的人文風情

**優化建議：**
1. 使用影片壓縮工具減少檔案大小
2. 確保影片為橫向格式
3. 避免過長的影片以免影響載入速度
4. 可考慮提供不同解析度版本供不同裝置使用
5. 建議搭配輕快的音樂營造夏日海風氛圍

**替代方案：**
如果暫時沒有影片，可以：
- 使用東石漁港或濕地的高品質靜態圖片替代
- 從免費影片庫下載相關素材（如 Pexels、Pixabay）搜尋 "fishing village", "oyster farm", "wetland"
- 製作東石風景的幻燈片動畫
