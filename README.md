# 東石鄉觀光旅遊網 - 網頁程式設計期末專題

## 專題簡介
這是一個以嘉義縣東石鄉觀光旅遊為主題的網站，透過「牡蠣寶寶」吉祥物帶領使用者探索東石鄉的海洋魅力、在地美食與濱海風光。

## 設計理念
- **主題色彩**：藍白海風配色，營造夏日沁涼感
- **吉祥物**：牡蠣寶寶（可愛的牡蠣角色），代表東石豐富的蚵養殖產業
- **視覺風格**：整體結合文字、影像與互動，塑造沉浸式瀏覽體驗

## 網站結構

### HTML 頁面
- `index.html` - 歡迎頁面（載入動畫，引導進入網站）
- `welcome.html` - 首頁（包含 Hero 影片、熱門景點、即時資訊）
- `about.html` - 東石特色（介紹東石之美與吉祥物）
- `events.html` - 在地美食（鮮蚵、蚵仔煎、海產小吃等）
- `routes.html` - 推薦路線（生態探索、美食探索、文化探索）
- `wheel.html` - 浪一浪轉盤（隨機景點推薦，增加趣味性）
- `map.html` - 互動地圖（顯示景點、美食、住宿）
- `contact.html` - 聯絡我們

### CSS 樣式（藍白海風主題）
- `css/style.css` - 全站基本樣式（藍白配色）
- `css/nav.css` - 導覽列（固定浮動設計）
- `css/home.css` - 首頁
- `css/about.css` - 東石特色頁面
- `css/events.css` - 美食頁面
- `css/routes.css` - 路線頁面
- `css/map.css` - 地圖頁面
- `css/wheel.css` - 轉盤頁面

### JavaScript 功能
- `js/main.js` - 全站共用功能（導覽列、通知系統）
- `js/weather-api.js` - 即時資訊（天氣、空氣品質、日落時間）
- `js/wheel.js` - 浪一浪轉盤功能
- `js/map.js` - 互動地圖功能（景點、美食、住宿分類）
- `js/animation.js` - 各種動畫效果

### 圖片資源
- `images/hero/` - 首頁主視覺
- `images/about/` - 東石特色圖片
- `images/events/` - 美食圖片
- `images/routes/` - 路線圖片
- `images/icons/` - 各種圖示
- `images/mascot/` - 牡蠣寶寶吉祥物

### 影片資源
- `videos/intro.mp4` - 首頁開場 Vlog（展示東石風光）

## 主要功能

### 1. 響應式導覽列
- 固定在頁面頂部（固定浮動模式）
- 支援手機版選單
- 目前頁面高亮顯示
- 細項選單展開功能

### 2. 即時資訊服務
- **天氣預報**：顯示東石當地天氣
- **空氣品質指標**：即時 AQI 資訊
- **日落時間**：適合觀賞夕陽的時間

### 3. 浪一浪轉盤
- Canvas 繪製的互動轉盤
- 隨機推薦東石景點
- 平滑旋轉動畫效果
- 增加旅遊趣味性

### 4. 互動地圖
- 東石地圖視覺化
- 可篩選類別（景點、美食、住宿）
- 點擊標記顯示詳細資訊
- 顯示小吃名店、住宿地點及交通路線

### 5. 推薦路線系統
- **生態探索路線**：鰲鼓濕地、漁港、蚵棚體驗
- **美食探索路線**：鮮蚵大餐、海產小吃、傳統市場
- **文化探索路線**：先天宮、漁村風情、地方特產
- 滿足不同族群的旅遊需求

### 6. 動畫效果
- 滾動淡入效果
- 滾動進度條
- 返回頂部按鈕
- 卡片懸停動畫
- 首頁自動播放影片

## 網站功能特色

### (一) 導覽設計
1. **頂端導覽列**：設置於頁面頂端，讓使用者能一目瞭然網站內容結構
2. **主題分類**：涵蓋東石特色、美食、路線等分類，方便主題導向瀏覽
3. **細項選單**：滑鼠指向導覽列時展開細項，協助使用者循序漸進找到所需資訊
4. **固定浮動模式**：首頁及頂端按鈕設計為固定浮動，滑動網頁時仍可隨時操作

### (二) 功能與互動設計
1. **即時資訊服務**：
   - 天氣預報
   - 空氣品質指標
   - 日落時間
2. **互動性功能**：
   - **互動地圖**：顯示小吃名店、住宿地點及交通路線
   - **浪一浪轉盤**：隨機推薦旅遊景點，增加趣味性
   - **推薦路線**：提供生態、美食、文化三大路線，滿足不同族群需求

### (三) 視覺與風格設計
1. **海風主題**：整體版面以「海風」為主題，色彩採用藍白配色，營造夏日沁涼感
2. **吉祥物設計**：「牡蠣寶寶」——可愛的牡蠣角色，陪伴使用者瀏覽網站，增添親和力
3. **首頁影片**：自動播放影片，提升地方印象與觀光吸引力
4. **美景展示**：輪播美景照片區，透過圖文並茂的方式展示東石風光
5. **沉浸式體驗**：整體結合文字、影像與互動，促進東石觀光發展

## 如何使用

### 1. 直接開啟
將專案資料夾放置到本機，使用瀏覽器開啟 `index.html`（會先顯示歡迎動畫，然後自動進入 `welcome.html` 首頁）

### 2. 使用本地伺服器（推薦）
```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js (需安裝 http-server)
npx http-server

# 使用 VS Code Live Server 擴充功能
```

然後在瀏覽器中開啟 `http://localhost:8000`
